# lumos

Test Project

## Installing:

**1. Clone this repo to your folder:**

```
git clone https://github.com/oleksandr-samoilenko-ew/lumos_test
```

**2. Change current directory to the cloned folder:**

```
cd lumos
```

**3. Get packages**

```
flutter pub get
```
